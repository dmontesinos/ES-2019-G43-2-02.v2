\section{TP-02 : Inicio de sesión}\label{sect:TP1}

En el siguiente caso de test se muestran los posibles inputs y outputs al realizar el login del usuario.


\TPStep{01}{Inicia del sistema}{El sistema se inicia sin ningún usuario por defecto. Previamente se debe haber creado manualmente en la base de datos con el usuario:contraseña root:toor}{El sistema arranca sin problemas}{}

\TPStep{02}{Inicio de sesión válido con contraseña válida : Usuario}{Se accede al login del sistema y se pide los datos de usuario: \textit{new\_user}}{El sistema pide la contraseña}{}

\TPStep{03}{Inicio de sesión válido con contraseña válida : Contraseña: Contraseña}{Se introduce la contraseña válida : \textit{CXX965D!}}{El sistema valida al usuario:
\begin{itemize}
\item  Redirige al usuario a su panel principal

\end{itemize}
}{}

\TPStep{04}{Inicio de sesión inválido con usuario inválido : Usuario}{Se introduce un usuario que no existe en la base de datos : \textit{inventado!}}{
\begin{itemize}
\item  El sistema comprueba en la base de datos y determina que no existe
\item Se devuelve un mensaje al usuario con el error
\end{itemize}
}{}

\TPStep{06}{Inicio de sesión inválido con contraseña inválida : Contraseña}{Se introduce un usuario y una contraseña que no coincide con la cuenta}{
\begin{itemize}
\item  El sistema comprueba en la base de datos y determina que no coincide el login con la contraseña
\item Se devuelve un mensaje al usuario con el error
\end{itemize}
}{}

\TPStep{07}{Inicio de sesión inválido con contraseña inválida: Control de
repeticiones}{Volvemos al paso 6 y lo repetimos 3 veces}{
\begin{itemize}
\item  El sistema va contando los inicios incorrectos y determina que debe bloquear el usuario por precaución
\item Envía al usuario un mensaje de que su cuenta ha quedado bloqueada por actividad sospechosa
\item Se envía un mensaje al administrador para que tome las medidas necesarias
\end{itemize}
}{}

\TPStep{08}{Inicio de sesión inválido con usuario bloqueado: Control de
repeticiones}{Se accede al registro y se determina que la cuenta introducida por el usuario está bloqueada}{El sistema devuelve al usuario un mensaje de error mostrándole los motivos}{}
